## 메인 작업 목표 (우선순위 순)

### 1. 영상 자막 생성 (STT → SRT → Premiere)
Premiere 자동자막 대신 Whisper 기반으로 정확도 높은 자막 생성.

**실행 명령어:**
```bash
whisper "영상파일경로.mp4" \
  --language Korean \
  --model medium \
  --output_format srt \
  --output_dir "출력폴더경로"
```

**이후:**
- 생성된 `.srt` 파일을 Premiere MCP `create_caption_track`으로 자막 트랙 자동 생성

**모델 선택 기준:**
- `small` : 빠름 (~5분/11분 영상), 정확도 보통
- `medium` : 균형 (~15분), 권장
- `large` : 최고 정확도 (~30분+), 긴 영상 주의

---

### 2. 자동 컷편집

**확정된 접근법: 파형(ffmpeg) + STT(SRT) 하이브리드**

```bash
# 무음 감지 (피아노 연주는 -35dB 이상이라 보존됨)
ffmpeg -i 음원.mp3 -af silencedetect=noise=-35dB:d=0.3 -f null - 2>&1 | grep silence
```

**컷 대상:**
1. **무음 구간** — ffmpeg silencedetect로 감지, 피아노 연주 구간은 음량이 있어 자동 보존
2. **필러/더듬거림** — SRT 텍스트에서 패턴 감지:
   - 단독 짧은 발화: "어", "음", "그", "저", "아", "이제", "뭐" 등
   - 불완전 문장 (말이 흐지부지 끊기는 구간)
   - 수정 발화: "아니", "다시", "그러니까" 로 시작하는 재시도 구간
   - 완전 중복 자막 (Whisper가 반복 감지한 구간)

**⚠️ 중요 한계 및 주의사항:**

| 항목 | 내용 |
|------|------|
| 무음 임계값 | ≥1.5초 이상만 auto-cut 권장. 0.8초는 너무 러프해서 결국 수동 수정 필요 |
| 필러 감지 정확도 | SRT 기반이라 Whisper 오인식 있음. 목록 확인 후 승인 방식으로 진행 |
| 연주 중 "이렇게" 등 | 연주 시범 중 짧은 설명어는 컷하면 안 됨 — 맥락 확인 필수 |

**razor 컷 후 V1+A1 링크 재연결 필수:**
- `razor_timeline_at_time`으로 컷하면 V1-A1 링크가 해제됨
- 컷 완료 후 `link_audio_video`로 각 클립 재링크 필요
- 또는: 컷 전 V1만 lock하고 A1만 컷 → 이후 수동 링크

**SRT와 컷편집 관계:**
- Lift delete (빈칸 유지): SRT 타임스탬프 그대로 유지
- Ripple delete (간격 제거): 타임스탬프 틀어짐 → 최종 수출 후 Whisper 재실행 필요

## 다음 세션 표준 워크플로우

### Step 1 — STT 자막 생성
```bash
whisper "음원.mp3" --language Korean --model medium --output_format srt --output_dir "자막/"
```

### Step 2 — Premiere에 자막 트랙 바로 열기
STT 완료 즉시 `create_caption_track`으로 Premiere에 SRT 임포트
→ 컷 리뷰 시 자막 보면서 편집 가능

### Step 3 — 컷 목록 추출
```python
# ffmpeg silencedetect (≥1.5초 기준)
ffmpeg -i 음원.mp3 -af silencedetect=noise=-35dB:d=0.3 -f null - 2>&1

# SRT 분석 — 감지 대상
# - 무음 ≥1.5초 (auto-cut 후보)
# - 무음 0.8~1.5초 (검토 목록)
# - 필러: "어", "음", "그", "저", "아", 단독 "이렇게/그냥/이런"
# - 불완전 문장 (트레일오프)
# - 수정 발화 (앞 말 취소 후 재시도)
# - Whisper 중복 감지 구간
```

### Step 4 — 컷 목록 사용자 확인 후 razor
목록 보여주고 **승인 후** 일괄 razor → 러프 편집 방지

### Step 5 — razor 후 V1-A1 링크 재연결
```
razor 완료 후: 컷된 각 클립에 link_audio_video 재호출
```

### Step 6 — 최종 수출 후 SRT 재생성 (ripple delete 사용 시)
```bash
whisper "최종수출본.mp4" --language Korean --model medium --output_format srt
```
